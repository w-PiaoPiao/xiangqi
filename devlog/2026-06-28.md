## 待办事项（明天继续）
- [ ] **棋子文字太小**：已调至 `radius * 1.9` 仍不够大，推测原因：
  - `ctx.font` 语法可能不被 HarmonyOS Canvas 正确解析 → 尝试用纯数字 + 指定中文字体 `'HarmonyOS Sans'`
  - 或 Canvas 坐标系统是 px 而视觉缩放后字偏小 → 改用 `vp2px` 或字号直接设为 `cellSize * 0.9`
- [ ] **没有棋子可走位置显示（绿色提示点不出现）**：
  - 根因：`GameView.onCellClick()` 中选中/取消选中/切换选中时没有调 `this.redrawTrigger++`，只走了 `syncBoard()` 的情况（走棋）才更新 → BoardCanvas 的 `@Watch('onRedraw')` 不会触发，`drawAll()` 不会重绘
- [ ] **没有棋子选中高亮显示（金色外圈不出现）**：
  - 同上面因，`selectedPos` @Prop 虽然变了，但没有 `@Watch` 触发，Canvas 不重绘
- [ ] **`@Watch` 回调用法**：`onRedraw()` 无参数，但 HarmonyOS 要求 `@Watch` 回调接收 `(propName: string)`，可能编译不报错但运行时异常
- [ ] 配置签名（signingConfigs）以生成可安装到真机的已签名 HAP
- [ ] 考虑实现桌面/大屏横屏布局（响应式 breakpoint）
