# 开发需求文档

## 1. 项目概述
轻量化本地中国象棋（象棋）应用，纯前端实现，浏览器直接打开即可运行。支持双人本地轮流对弈。

## 2. 功能需求

### 2.1 核心玩法
- [ ] 标准 9×10 棋盘，完整棋子布局（红方在下，黑方在上）
- [ ] 红黑双方轮流走棋，红方先手
- [ ] 全部七种棋子的合法走法判定：
  - 将/帅：九宫内一步直行
  - 士/仕：九宫内一步斜行
  - 象/相：田字对角行走，塞象眼判定
  - 马：日字行走，蹩马腿判定
  - 车：直线任意步数
  - 炮：直线移动、翻山吃子
  - 兵/卒：过河前只能前进，过河后可横移
- [ ] 吃子判定
- [ ] 将军（check）检测与提示
- [ ] 将帅对面（flying general）规则判定
- [ ] 将杀（checkmate）检测与胜负判定
- [ ] 困毙（stalemate）检测

### 2.2 交互功能
- [ ] 点击棋子选中并高亮
- [ ] 显示选中棋子的合法走法（绿色圆点提示）
- [ ] 点击合法目标执行走棋
- [ ] 点击己方其他棋子切换选中
- [ ] 当前回合信息显示
- [ ] 将军警告提示

### 2.3 辅助功能
- [ ] 重新开始（新对局）
- [ ] 悔棋（撤销上一步）
- [ ] 走棋历史记录（代数式或中文描述）

## 3. 非功能需求
- **零依赖**：纯 HTML/CSS/JavaScript，无任何外部库或框架
- **单文件部署**：仅 index.html，双击即可运行
- **离线可用**：不需要网络连接
- **响应式**：在不同屏幕尺寸下均可正常显示
- **性能**：棋盘渲染和走法计算流畅无卡顿
