# UI/UX 设计规范

## 1. 布局

```
┌────────────────────────────────┐
│          中国象棋               │  ← 标题栏
├────────────────────────────────┤
│                                │
│        棋盘 (Canvas)           │  ← 主要游戏区域
│        约 400×440px            │
│                                │
├────────────────────────────────┤
│  当前：红方走棋    ⚠️ 将军！    │  ← 状态栏
│  [重新开始]  [悔棋]             │  ← 操作按钮
│  ──────────────────────        │
│  炮二平五  马8进7               │  ← 走棋历史列表
│  马二进三  车9平8               │
│  ...更多记录...                 │
└────────────────────────────────┘
```

## 2. 色彩方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 棋盘底色 | `#DEB887` (burlywood) | 暖木色 |
| 网格线 | `#4A3728` | 深棕色 |
| 红方棋子 | `#C0392B` (填充) | 红色 |
| 黑方棋子 | `#2C3E50` (填充) | 深蓝黑 |
| 棋子文字 | `#FFFFFF` | 白色 |
| 选中高亮 | `#F1C40F` (金黄) | 外圈 |
| 合法走法点 | `#27AE60` (半透明绿) | 圆点 |
| 历史文字 | `#333333` | 深灰 |

## 3. 字体
- **中文棋子文字**：STKaiti / 楷体，粗体
- **界面文字**：系统无衬线字体 (`-apple-system, sans-serif`)
- **历史记录**：等宽字体 (`monospace`)

## 4. 棋子样式
- 圆形棋子，直径约 38px
- 内圈圆形边框装饰
- 中间显示单个汉字（帅/将、仕/士、相等）
- 红方：红色填充 + 白色文字 + 浅色内圈
- 黑方：深色填充 + 白色文字 + 浅色内圈

## 5. 交互反馈
| 操作 | 反馈 |
|------|------|
| 选中己方棋子 | 金色外圈 + 绿色合法走法点 |
| 执行走棋 | 棋盘重绘 |
| 将军 | 状态栏文字变为红色 + "⚠️" 提示 |
| 将杀/困毙 | 弹窗或醒目文字提示胜负 |
| 点击非法位置 | 无反应（或取消选中） |

## 6. 响应式设计
- 棋盘尺寸根据屏幕宽度自适应缩放
- 最小宽度：360px
- 最大宽度：600px
- 使用 CSS `max-width` + `margin: auto` 居中
