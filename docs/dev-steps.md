## 开发原则
1. **逐阶段推进**：完成一个阶段并验证后，再进入下一阶段
2. **每步可验证**：每个步骤都有明确的验证标准
3. **先核心后外围**：先实现核心游戏逻辑，再完善 UI 辅助功能
4. **日志记录**：每次开发后在 devlog/ 中记录完成事项和待办事项
